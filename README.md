# 🏥 Hospital Patient Readmission Prediction

**Tech Stack:** Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn, SQL

## 📌 Overview
End-to-end supervised ML pipeline to predict hospital patient readmission 
probability across multi-demographic patient records.

## 🔍 Key Highlights
- Data cleaning, validation, and missing value handling
- EDA: diagnosis distribution by age group, effect of diabetes on readmission
- Trained Random Forest and Decision Tree classifiers
- Identified circulatory conditions as the most prevalent primary diagnosis
